split-by
========

Split an array or a string by another array or string.

example
=======

``` js
```

methods
=======

``` js
var splitBy = require('split-by')
```

splitBy(xs, ys)
---------------

Return an array of `xs` split by `ys`, not including the elements split on.

install
=======

With [npm](http://npmjs.org) do:

```
npm install split-by
```
